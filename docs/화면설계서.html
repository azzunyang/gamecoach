시스템 개요

서비스 개요

GameCoach는 게임 코치(강사)와 수강생을 연결하는 1:1 코칭 마켓플레이스입니다. MetaMask 지갑 인증을 통해 가입하고, Ethereum 스마트 컨트랙트 에스크로로 결제 신뢰성을 보장합니다.

사용자 역할
역할설명주요 화면
Guest 비회원미로그인 사용자랜딩, 코치 목록/상세, 로그인, 회원가입
Student 수강생코치에게 수업 신청코치 목록/상세, 수강생 대시보드, 채팅, 리뷰

API

API 명세

인증 API
GET /api/auth/nonce Nonce 발급

쿼리: ?address={wallet_address}

응답: { nonce: string } 또는 { error: string }

UUID nonce 생성 → KV에 nonce:{address} 키로 5분 TTL 저장

POST /api/auth/wallet 로그인 / 회원가입 
{
  "address": "0x...",
  "signature": "0x...",
  "role": "coach" | "student"  // 회원가입 시만
}
처리 단계설명
1. Nonce 검증KV에서 조회, 만료 시 401 반환, 사용 후 즉시 삭제
2. 서명 검증ethers.verifyMessage(message, signature) 복원 주소 비교
3a. 로그인D1 users에서 address 조회, 없으면 404
3b. 회원가입D1에 신규 유저 INSERT (id, role, wallet)
4. 세션 생성KV에 session:{token} 저장 (7일)
5. 쿠키 설정gc_session HttpOnly 쿠키
GET /api/auth/me 현재 세션 정보

인증: gc_session 쿠키 필요. 미인증 시 401.

// 응답
{
  "id": "uuid",
  "address": "0x...",
  "role": "coach" | "student",
  "nickname": "string"
}
POST /api/auth/logout 로그아웃

KV에서 세션 삭제 → gc_session 쿠키 만료 (Max-Age=0)

응답: { ok: true }

코치 API
GET /api/coaches 코치 목록

쿼리: ?category={categoryId} (선택) → { coaches: Coach[] }

GET /api/coaches/[id] 코치 상세

코치 상세 객체 반환

GET /api/coaches/[id]/slots 예약 가능 슬롯

{ slots: string[] } — 예약 가능한 날짜/시간 목록

수업 API
GET /api/lessons 수업 목록

인증 필요. 응답: { lessons: Lesson[] } (본인 관련 수업만)

Lesson 상태값: PENDING | ACCEPTED | ACTIVE | COMPLETED | REJECTED

POST /api/lessons 수업 신청 
{ "coach_id": "string", "slot": "2026-06-09 18:00", "message": "string", "goal": "string" }
PATCH /api/lessons/[id] 수업 상태 변경 (코치) 
{ "state": "ACCEPTED" | "REJECTED" }

해당 수업의 전체 메시지 반환

POST /api/messages/[lessonId] 메시지 전송 
{ "body": "string" }
