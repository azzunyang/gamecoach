서비스 개요

GameCoach 란?

🎮 핵심 가치

게임 실력 향상을 원하는 플레이어와 검증된 고티어 코치를 연결하는 1:1 코칭 마켓플레이스

🔗 Web3 차별점

기존 결제 대신 스마트 컨트랙트 에스크로를 사용해 수업료를 안전하게 보관하고, 완료 후 자동 정산

1

코치 탐색

게임 · 티어 · 가격으로 필터링, 강의 커리큘럼과 후기 확인

2

슬롯 예약 & 선납

원하는 날짜 선택, ETH 30% 예약금을 스마트 컨트랙트에 예치

3

1:1 Discord 코칭

음성 + 화면 공유로 실시간 맞춤 코칭 진행

4

완료 확인 & 자동 정산

학생이 완료 버튼 → 잔금 포함 전액이 코치 지갑에 자동 전송

주요 기능

플랫폼이 제공하는 것들

🔍

코치 탐색 & 강의

게임·카테고리·별점·가격으로 필터링. 코치별 강의 목록, 커리큘럼, 수강 후기 열람

💳

MetaMask 지갑 인증

전화번호 OTP + MetaMask 서명으로 본인 확인. 지갑 주소 기반 신원 보장

📅

슬롯 예약 시스템

코치가 가능 시간대를 등록하면 학생이 선택. 중복 예약 방지

💬

수업 내 채팅

예약 완료된 수업 전용 채팅방 제공. 코치·학생 간 소통 및 일정 조율

⭐

검증 리뷰 시스템

수업 완료 학생만 후기 작성 가능. 설명·소통·시간·커리큘럼 4항목 평가

🛡️

관리자 대시보드

코치 승인·정지, 신고 처리, 분쟁 해결. 플랫폼 통계 모니터링

블록체인 활용

왜 블록체인이 필요한가?

코치가 수업 후 돈을 받지 못하거나, 학생이 수업 전 환불을 못 받는 신뢰 문제를 코드로 해결

📌 네트워크: Ethereum Sepolia Testnet

실제 이더리움과 동일한 구조의 테스트넷. 메인넷 배포 전 검증용

🔐 에스크로 구조

• 예약 시 30% 선납 → 컨트랙트 잠금
• 수업 시작 전 70% 추가 납부
• 수업 완료 후 100% 코치 지갑으로 자동 전송
• 코치 거절 시 즉시 자동 환불

스마트 컨트랙트

GameCoachEscrow.sol

함수호출자동작
requestLesson 학생 30% ETH 예치, 수업 생성
acceptLesson 코치 수업 수락 → ACCEPTED
rejectLesson 코치 거절 → 예약금 즉시 환불
payBalance 학생 잔금 70% 추가 예치
confirmCompletion 학생 전액 코치 지갑으로 전송
cancelLesson 학생 취소 → 전액 환불
resolveDispute 관리자 분쟁 판정 → 승자에게 전송

💡 에스크로의 신뢰 보장

• 결제금은 플랫폼이 아닌 컨트랙트가 보관
• 누구도 임의로 빼갈 수 없음
• 모든 거래 이력이 블록체인에 영구 기록
• 조건 충족 시 코드가 자동 실행
