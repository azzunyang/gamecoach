# GameCoach 화면 설계서

## 전체 화면 흐름

```
[미인증]
홈(/) ──→ 코치목록(/coaches) ──→ 코치상세(/coaches/[id])
  │                                      │
  │                              [수업 신청 모달]
  │                              달력 → 수업안내 → 결제확인
  │
  ├──→ 로그인(/auth/login)
  │         │ 성공
  │         ↓
  │        홈(/)
  │
  └──→ 회원가입(/auth/register)
            │ 성공
            ↓
       프로필설정(/profile/setup)
            │ 저장 or 건너뛰기
            ↓
           홈(/)

[인증 후]
홈(/) ──→ 대시보드(/dashboard/student | /dashboard/coach)
              │
              ├──→ 채팅(/chat/[lessonId])
              └──→ 리뷰작성(/review/[lessonId])
```

---

## 공통 컴포넌트

### GNB (상단 네비게이션)
- 좌측: `🎮 GameCoach` 로고 (홈 링크)
- 우측: `AuthNav` 컴포넌트
  - **비로그인**: `로그인` 버튼 + `회원가입` 버튼
  - **로그인됨**: `🦊 0x1234...abcd` + `대시보드` 버튼 + `로그아웃` 버튼

### SubNav (서브 네비게이션)
홈 · 강의 목록 · 코치 목록 · 후기 · 찜 목록

- 현재 경로에 맞게 탭 활성화 (파란 언더라인)
  - `/` → 홈
  - `/coaches*` → 코치 목록
  - `/dashboard*` → 찜 목록
- 적용 페이지: 홈, 코치 목록, 코치 상세

## 화면별 설계

### 1. 홈 `/`

**목적**: 플랫폼 소개 및 코치·강의 탐색 진입점

**레이아웃**
```
┌─────────────────────────────────────┐
│ GNB                                 │
├─────────────────────────────────────┤
│ SubNav                              │
├─────────────────────────────────────┤
│ 히어로 (다크 그라디언트)             │
│   게임 고수에게 배우는 1:1 코칭       │
│   검증된 코치 · 블록체인 에스크로     │
├─────────────────────────────────────┤
│ 게임 카테고리 아이콘 (가로 스크롤)    │
│  🎯FPS  ⚔️MOBA  🧩전략  🦸팀파이트   │
│  🪂배틀로얄  ⭐캐주얼  🎮기타        │
├─────────────────────────────────────┤
│ 지금 수강 가능한 코치 (N명)  전체보기→│
│ [카드] [카드] [카드] ...            │
├─────────────────────────────────────┤
│ 새로운 강의 (8개)          전체보기→ │
│ [카드] [카드] [카드] ...            │
└─────────────────────────────────────┘
```

**인터랙션**
- 카테고리 아이콘 클릭 → 코치 목록 필터링 (토글)
- 코치 카드 클릭 → `/coaches/[id]`
- 강의 카드 클릭 → `/coaches/[id]`
- "전체보기 →" → `/coaches`

**카드 구성 (코치)**
- 썸네일: 카테고리별 그라디언트 + 아이콘 + 닉네임 + 티어
- 바디: 소개 2줄, ETH 가격, 별점

**카드 구성 (강의)**
- 썸네일: 카테고리별 그라디언트 + 제목 + 시간
- 바디: 코치명, 수강생 수, ETH 가격, 별점

### 4. 로그인 `/auth/login`

**목적**: MetaMask 지갑으로 로그인

**레이아웃**
```
┌───────────────────────┐
│ ← 홈으로  🎮GameCoach │
│                       │
│ 로그인                │
│ MetaMask 지갑으로     │
│ 간편하게 로그인하세요  │
│                       │
│ [🦊 MetaMask로 로그인]│
│                       │
│ 💡 MetaMask 설치 안내 │
│                       │
│ 처음이신가요? [회원가입]│
└───────────────────────┘
```

**플로우**
1. 버튼 클릭 → MetaMask 지갑 연결 팝업
2. 지갑 연결 → nonce 발급 → 서명 요청 팝업
3. 서명 완료 → 서버 검증
   - 기존 사용자 → 홈(`/`)
   - 미등록 지갑 → 에러 + 회원가입 링크

**상태 표시**
- `지갑 연결 중...` (spinner)
- `MetaMask에서 서명해주세요...` (spinner)
- 에러 시 빨간 알림 박스

## DB 테이블 요약

| 테이블 | 주요 컬럼 | 비고 |
|--------|-----------|------|
| `users` | id, wallet(UNIQUE), phone?, role, nickname, created_at | coaches.id = users.id |
| `coaches` | id(=users.id), nickname, game_category, tier, price_eth, session_min, intro, avg_rating | 코치 전용 프로필 |
| `slots` | id, coach_id, date, start_time, end_time, is_booked | 예약 가능 시간 |
| `lessons` | id, coach_id, student_id, slot_id, state, deposit_eth, balance_eth | 수업 상태 머신 |
| `messages` | id, lesson_id, sender_id, body, created_at | 채팅 |
| `reviews` | id, lesson_id, coach_id, student_id, score_*, body | 리뷰 |

## 수업 상태 흐름

```
PENDING(예약요청)
  ├─→ ACCEPTED(수락) ─→ ACTIVE(수업중) ─→ COMPLETED(완료)
  │                  └─→ CANCELLED
  └─→ REJECTED(거절)
                         ACTIVE ─→ DISPUTED(분쟁) ─→ RESOLVED
```
